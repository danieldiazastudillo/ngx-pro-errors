ngx-pro-errors

This is a third iteration of @ultimate/ngxerrors. We don't know why but UltimateAngular never updated this package to be used with Angular 6 (when rxjs was updated to use pipeable operators). Enters @hackages/ngx-errors who updated the project and most of it's dependencies so we could use it in our projects as usual

Test Form

Very simple form. If you want to try more features clone the repo and run the showcase

The original project was using some custom Webpack, Rollup & Yarn configuration and Angular CLI was nowhere to be seen. This is the same project as @hackages but using the current CLI tools so we can take advantage of update schematics and a more aligned design with what Angular is right now (meaning Ivy). If you are using the original project, usage instructions are the same, nothing has been changed. Good luck!
